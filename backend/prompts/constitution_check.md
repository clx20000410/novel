## 检查维度

请逐一检查以下维度，并给出合规性评估：

### 1. 故事基础合规
- 内容是否服务于核心主题？
- 是否符合故事类型？
- 是否与核心冲突一致？
- 是否符合故事走向？
- 是否传达核心价值观？

### 2. 叙事视角合规
- POV 类型是否正确？
- 是否遵守 POV 限制规则？
- 是否泄露了 POV 角色不应知道的信息？
- POV 切换是否合理标记？

### 3. 目标受众合规
- 内容分级是否合适？
- 语言水平是否适合目标读者？
- 主题是否适合目标年龄？

### 4. 风格基调合规
- 整体基调是否一致？
- 语言风格是否匹配？
- 情感基调是否合适？

### 5. 世界观合规
- 是否尊重世界类型设定？
- 是否遵守力量/魔法系统规则？
- 是否触犯禁忌内容？
- 世界观是否一致？

### 6. 角色约束合规
- 角色类型是否允许？
- 角色能力是否在上限内？
- 角色行为是否合理？

### 7. 剧情约束合规
- 剧情类型是否允许？
- 转折频率是否合适？
- 伏笔规则是否遵守？

### 8. 时空约束合规
- 是否在时间跨度内？
- 是否在地理范围内？
- 时间流动是否合规？

## 输出格式

请以 JSON 格式输出检查结果：

```json
{
  "overall_compliance": true/false,
  "overall_score": 0-100,
  "violations": [
    {
      "dimension": "维度名称",
      "severity": "critical/warning/info",
      "description": "违规描述",
      "location": "违规位置（段落/句子）",
      "suggestion": "修复建议"
    }
  ],
  "summary": "总体评估摘要"
}
```

## 严重性定义

- **critical**：严重违反宪法核心规则，必须修复
- **warning**：违反宪法建议规则，建议修复
- **info**：轻微偏离，可选修复
